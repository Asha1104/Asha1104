### Angelus School Receipt System - User Guide

Welcome to the Angelus School Receipt System! This application helps staff generate and manage receipts for tuition payments.

---

#### **Getting Started**

1. **Open the Application**  
   - Locate and run the file: `Angelus_Receipt_System.exe` (or `receipt_app.py` if running directly from Python).
   - The main screen will display a welcome message and a button to proceed.

2. **Using the Application**  
   - Click the **"Tuition Fee Receipt"** button on the main menu to start creating a receipt.
   - Fill in the required fields:
     - **Parent/Guardian Name**
     - **Childâ€™s Name**
     - **Class**
     - **Paid Amount (K)**
     - **Balance Left (K)**
     - **Payment Type** (Cash or Cheque)
     - **Served By** (Select your name from the dropdown)
   - Click **"Calculate & Preview"** to review the receipt.

---

#### **Saving & Printing Receipts**

1. **Save to PDF**  
   - Each receipt is automatically saved as a PDF in the `receipts` folder within the application's directory.

2. **Print the Receipt**  
   - After previewing the receipt, click **"Print"** to send it to the default printer.

---

#### **Other Features**

- **Clear the Form**: Use the **"Clear"** button to reset all fields.
- **Go Back**: Click **"Back"** to return to the main menu.

---

#### **Troubleshooting**

1. **Missing `school_logo.png`**  
   Ensure the logo file is in the same directory as the application.

2. **Cannot Save or Print Receipts**  
   - Ensure you have permission to create files in the `receipts` folder.
   - Check that the default printer is correctly configured.
